# Student Grade Calculator

## Description
A simple Python program that calculates a student's grade based on marks entered by the user.

## Features
- Input validation (0-100 marks only)
- Uses functions
- Uses if-elif-else statements
- Uses while loop for invalid inputs
- Displays encouraging messages

## Grading System

| Marks | Grade |
|---------|---------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 0-59 | F |

1. Open terminal
2. Navigate to project folder
